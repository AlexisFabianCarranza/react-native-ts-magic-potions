## Step by step

- Environtment setup: <p> 
https://reactnative.dev/docs/environment-setup
- Clone the repository: <p> 
`git clone https://github.com/AlexisFabianCarranza/react-native-ts-magic-potions.git`
- Install dependencies: <p> 
`yarn`
- Run project: 
`npm run android`
`npm run ios`
- Run test:
`yarn test`
